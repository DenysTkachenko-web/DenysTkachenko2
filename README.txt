Инструкции по использованию / Замена содержимого
-----------------------------------------------
Файлы в архиве:
- index.html             — основной одностраничный сайт с секциями "Обо мне" и "Сравнение дронов".
- css/style.css          — стили оформления (подключается в index.html).
- js/main.js             — скрипт для переключения секций и интерактивного сравнения дронов.
- data/drones.json       — данные с информацией о дронах (JSON). Можно заменить своими данными.
- README.txt             — этот файл.

Как использовать:
1) Распакуйте архив и откройте index.html в браузере.
2) Для изменения CV откройте секцию <section id="section-about"> в index.html и замените текст.
3) Для замены сравнения дронов — отредактируйте data/drones.json (массив объектов, ключи: model,camera,flight,weight,price,rating).
4) Если хотите встроить локальный готический шрифт — поместите файлы шрифтов в папку css/fonts/ и подключите через @font-face в css/style.css.
